title: "静默与抑制"
---

## 静默

静默用于在计划内维护窗口期间暂停告警通知，**但告警仍会正常记录和状态流转**。

| 配置项   | 说明                     |
| -------- | ------------------------ |
| 目标     | 应用的对象               |
| 级别     | 过滤的特定级别           |
| 规则     | 过滤的特定规则           |
| 开始时间 | 静默生效时间             |
| 结束时间 | 静默失效时间             |
| 重复周期 | 静默将会以此值为周期运行 |
| 备注     | 可选的说明               |

## 抑制

抑制用于减少特定源告警触发时关联的目标告警。

| 配置项 | 说明                                 |
| ------ | ------------------------------------ |
| 源     | 触发抑制的告警                       |
| 目标   | 被抑制的告警                         |
| 维度   | 源和目标必须处于相同维度才会触发抑制 |

<Info>已被静默的告警不会再应用机制规则。</Info>
