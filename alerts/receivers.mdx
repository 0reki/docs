---
title: "接收器"
---

接收器是告警通知的发送渠道。规则必须唯一对应一个接收器。

## 邮件

| 配置项   | 说明                                   |
| -------- | -------------------------------------- |
| 收件人   | 邮件地址列表，可通过逗号分隔以配置多个 |
| 主题模板 | 邮件标题模板                           |
| 正文模板 | 邮件正文模板                           |

## Webhook

| 配置项   | 说明                         |
| -------- | ---------------------------- |
| URL      | 接收通知的接口地址           |
| 方法     | `GET` 或 `POST`              |
| 请求头   | 自定义的 HTTP 请求头         |
| 正文模板 | 请求体内容，可使用模板变量。 |

## 企业微信

| 配置项      | 说明                            |
| ----------- | ------------------------------- |
| Webhook URL | 企业微信群机器人的 Webhook 地址 |
| 消息模板    | Markdown 格式消息，支持模板变量 |

## 通知模板变量

以下变量可用于邮件主题、正文及 Webhook 模板中：

| 变量                | 说明             |
| ------------------- | ---------------- |
| `{{.level}}`        | 告警级别         |
| `{{.title}}`        | 告警规则名称     |
| `{{.device}}`       | 主机名           |
| `{{.device_ip}}`    | 主机 IP          |
| `{{.metric}}`       | 触发告警的指标名 |
| `{{.value}}`        | 当前指标值       |
| `{{.threshold}}`    | 阈值             |
| `{{.triggered_at}}` | 触发时间         |
| `{{.duration}}`     | 持续时长         |
