---
title: "概述"
---

CloLens 用于记录和分析跨服务的请求链路。

## 指标

页面顶部展示当前过滤范围内的汇总数据：

| 指标     | 说明                       |
| -------- | -------------------------- |
| 调用链   | 显示的链路数量             |
| 异常     | 状态为错误或超时的链路数量 |
| P95 耗时 | 95% 分位耗时 / 平均耗时    |
| Span     | 所有链路的 Span 总数       |

## 状态

| 状态 | 说明                           |
| ---- | ------------------------------ |
| 正常 | 所有 Span 均成功完成           |
| 错误 | 有 Span 返回错误或携带错误信息 |
| 超时 | 链路耗时超过阈值               |
| 未知 | 无法确定的状态                 |
