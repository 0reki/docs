---
title: "设备详细信息"
---

点击设备卡片即可查看设备详细信息。

## 基础信息

| 信息     | 说明                                             |
| -------- | ------------------------------------------------ |
| 网络信息 | 设备名、IP、公网 IP                              |
| 系统信息 | 操作系统、发行版、内核版本、架构                 |
| 硬件信息 | CPU 型号、CPU 核数、内存容量、启动时间和运行时长 |

## 监控数据

包括 CPU、内存、磁盘、网络等实时监控数据

| 监控项  | 说明               |
| ------- | ------------------ |
| 流量    | 按网卡查看收发流量 |
| 磁盘 IO | 按磁盘查看读写数据 |

## 服务列表

当前设备上运行中，且被插件命中的服务。

## 删除设备

在设备详情页点击删除按钮即可删除当前设备。

<Warning>
  删除设备会同时删除该设备、服务、指标和相关运行数据，此操作不可恢复。
</Warning>
